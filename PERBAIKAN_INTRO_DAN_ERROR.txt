PERBAIKAN YANG DILAKUKAN

1. Struktur package Java sudah disesuaikan dengan folder core dan modes.
2. Import yang tidak dipakai dibersihkan agar warning editor berkurang.
3. Intro dibuat lebih sederhana dan stabil: MediaPlayer dibuat setelah Stage tampil,
