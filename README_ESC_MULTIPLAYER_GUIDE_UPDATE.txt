UPDATE ESC + PANDUAN MULTIPLAYER

1. Tombol MENU di dalam mode permainan dihapus.
2. Endless menampilkan petunjuk: TEKAN ESC UNTUK KELUAR.
3. Tournament tidak lagi memakai tombol MENU; ESC langsung kembali ke menu utama.
4. Multiplayer menampilkan panduan satu per satu sampai gol pertama:
   - Langkah 1: Penendang menarik bola dan melepasnya.
   - Langkah 2: Keeper memilih arah dengan klik di dalam area gawang.
   - Langkah 3: Menunggu hasil tendangan.
5. Jika bola meleset atau ditepis, panduan diulang pada ronde berikutnya sampai ada gol.
6. Setelah gol pertama, panduan selesai dan match berjalan normal.
7. Tekan ESC untuk keluar dari mode dan kembali ke menu utama.
